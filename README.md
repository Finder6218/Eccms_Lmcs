# Eccms_Lmcs
 《电费管理系统-阶梯月收费》是一款通用性极强、功能极强大、内容完整的管理系统，根据收费的流程与应用，而实际研发的阶梯月收费专业软件，该软件的功能主要包括： 1、基本信息（设置年月、基本信息和电表查询（按台区查询、按用户查询））； 2、抄表收费管理（抄表导入、抄表录入、收费录入、某月未交费用户查询、每日收票员合计、每月收费员合计、台区电费累计、期间收费统计、每月所有用户实际用电及金额合计和每月每台区用户实际用电及金额合计）； 3、历史数据查询（历史抄表查询（按年月查询台区抄表情况、按用电户查询抄表情况）和历史收费查询（按用电户查询收费情况、按年月查询台区收费情况））； 4、用电统计分析（按台区对比分析、按用电类型统计分析、按年月对比分析、同期用电对比分析）； 5、基础设置（台区设置、电价类别设置）
